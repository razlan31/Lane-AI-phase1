# Lane AI

Your Business Co-Pilot.

Project structure:
- src/ — app code (React + Tailwind + shadcn/ui)
- docs/ — pricing, runbook, safeguards
- branding/ — logos & theme
- landing/ — static site
- testdata/ — sample JSON

Run locally:
1) npm install
2) npm run dev

Build:
- npm run build

Docs: see the docs/ folder.
